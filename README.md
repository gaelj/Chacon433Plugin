# Domoticz Chacon Control plugin

Python plugin to interface a 433MHz board from a Raspbery Pi, and trigger some Chacon relays

## Installation

### WiringPi + 433Utils

<https://github.com/ninjablocks/433Utils>

### HCC

```
unzip hcc.zip
rm hcc.zip 
cd hcc
sudo chmod 4777 radioEmission
```
